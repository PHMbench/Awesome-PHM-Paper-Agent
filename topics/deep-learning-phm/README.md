## 📖 主题概述

深度学习技术在PHM领域的应用已成为当前研究热点，涵盖故障诊断、剩余寿命预测、健康状态评估等多个方面。该技术通过多层神经网络自动学习复杂的非线性特征，显著提升了传统机器学习方法的性能。

### 🎯 研究方向

#### 1. 故障诊断 (Fault Diagnosis)
- **CNN应用**: 图像化振动信号分析
- **RNN/LSTM**: 时序信号建模
- **自编码器**: 无监督异常检测
- **GAN**: 数据增强和故障模拟

#### 2. 剩余寿命预测 (RUL Prediction)
- **时序预测模型**: LSTM, GRU, Transformer
- **退化建模**: 物理-数据驱动混合方法
- **不确定性量化**: 贝叶斯深度学习

#### 3. 健康状态评估 (Health Assessment)
- **多模态融合**: 视觉、听觉、振动信号
- **迁移学习**: 跨域知识迁移
- **联邦学习**: 分布式健康监测
